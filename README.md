# Pokedex Nacional (Vanilla JS)

Aplicación web para consultar información detallada de Pokémon consumiendo la [PokeAPI](https://pokeapi.co/). El proyecto se centra en el renderizado dinámico y la gestión de asincronía sin frameworks externos.

## 🚀 Funcionalidades
- **Navegación:** Paginación de 20 en 20 para optimizar la carga de datos.
- **Filtros:** Buscador por nombre/ID, filtro por tipos y lista de favoritos.
- **Detalle Técnico:** Consulta de stats base, movimientos y descripción oficial en español.
- **UX Dinámica:** - Ciclo de sprites (animación) al hacer hover sobre las tarjetas.
  - Temas visuales automáticos según el tipo elemental.
  - Soporte para **Modo Oscuro** con persistencia en `localStorage`.

## 🛠️ Stack
- **Lenguaje:** JavaScript ES6+ (Vanilla).
- **Estilos:** CSS3 (Custom Properties para temas).
- **API:** PokeAPI v2.
